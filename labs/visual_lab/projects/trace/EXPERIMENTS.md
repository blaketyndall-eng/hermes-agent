# TRACE Experiment Index

This file tracks active and planned TRACE imaging experiments.

The purpose of each experiment is to isolate a specific visual behavior.

Avoid combining too many variables at once.

---

# Experiment 001 — Shadow

Status: active

Goal:
Study how shadow treatment changes atmosphere, believability, and phone avoidance.

Variables:
- lifted shadows
- shadow crush
- shadow texture
- balanced preservation

Outputs:
- comparison exports
- contact sheets
- CSV logs
- rubric scoring

---
