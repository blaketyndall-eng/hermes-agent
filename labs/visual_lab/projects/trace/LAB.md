# TRACE Imaging Lab

TRACE is a slim, screenless, daily-carry camera system focused on believable atmosphere and memory realism.

TRACE should preserve what phone cameras erase.

## Core research areas

- shadow behavior
- low-light realism
- texture preservation
- mixed-light color behavior
- roll coherence
- subtle blur
- compression realism
- sensor character

## Non-goals

Do not:

- simulate fake film overlays
- add scratches or sprockets
- imitate disposable-camera clichés
- build obvious retro presets
- destroy images for style

## Experiment flow

1. isolate a variable
2. generate comparisons
3. create contact sheets
4. score outputs
5. log observations
6. define next experiment
