# TRACE Next Steps

## Immediate priorities

### 1. Integrate TRACE_LAB prototype

Add the existing TRACE Imaging Lab repo into:

```text
labs/visual_lab/projects/trace/TRACE_LAB/
```

Preserve the standalone structure.

Do not prematurely refactor.

---

### 2. Build Experiment 001 dataset

Collect:

- low-light phone photos
- indoor mixed-light photos
- dark exterior photos
- accidental handheld images
- emotionally ordinary scenes

Avoid:

- cinematic photography
- curated influencer imagery
- highly edited references

---

### 3. Generate first scoring pass

Run Experiment 001 outputs through the Visual Lab rubric.

Document:

- strongest shadow treatment
- weakest treatment
- recurring artifacts
- unexpected behavior

---

### 4. Build contact sheet review workflow

The contact sheet should become the primary review surface.

Review should emphasize:

- emotional feel
- physical light behavior
- subtlety
- image memory quality

Not pure technical sharpness.

---

### 5. Define TRACE camera identity carefully

TRACE should eventually have recognizable behavior.

But:

It should not feel like a preset.

Research should focus on:

- stable tendencies
- not exaggerated signatures

---
