# TRACE Core Primitives

TRACE should be built from small behavioral primitives.

Not presets.
Not aesthetics.
Not monolithic filters.

A primitive is a reusable visual behavior that can be:

- observed
- adjusted
- combined
- evaluated
- compared historically

---

# 1. darkness_confidence

## Definition

The willingness of the imaging system to preserve darkness and partial visibility.

## Low values

- lifted shadows
- phone-like visibility
- flattened atmosphere

## High values

- dimensional darkness
- stronger atmosphere
- hidden information

## Failure mode

- muddy blacks
- crushed detail
- unusable shadow collapse

---

# 2. shadow_texture

## Definition

Subtle instability and physical texture within darker regions.

## Goal

Preserve tactile shadow behavior without visible fake grain.

## Failure mode

- obvious texture
- overlay feeling
- stylized noise

---

# 3. environmental_ambiguity

## Definition

Preservation of conflicting or unstable environmental lighting.

Examples:

- sodium vapor + fluorescent spill
- red neon + darkness
- tungsten + street light

## Goal

Avoid aggressive normalization.

## Failure mode

- muddy color confusion
- unrealistic color collapse

---

# 4. physical_falloff

## Definition

Believable spatial brightness decline across a frame.

## Goal

Create environmental depth and light realism.

## Failure mode

- fake vignette feeling
- obvious edge manipulation

---

# 5. foreground_pressure

## Definition

Visual dominance of foreground subjects or objects.

## Goal

Increase emotional immediacy.

## Failure mode

- background disappearance becoming stylized
- unnatural isolation

---

# 6. motion_softness

## Definition

Subtle blur or instability caused by handheld capture or low light.

## Goal

Preserve believable human capture presence.

## Failure mode

- unusable blur
- artificial softness

---

# 7. circulation_residue

## Definition

Subtle degradation from digital circulation.

Examples:

- screenshots
- repost chains
- compression softness
- resized uploads

## Goal

Preserve believable digital history.

## Failure mode

- gimmick degradation
- fake archive feeling

---

# 8. roll_drift

## Definition

Small behavioral variation across a sequence of images.

## Goal

Create coherence without uniformity.

## Failure mode

- random inconsistency
- obvious preset repetition

---

# 9. memory_anchor

## Definition

A visually dominant element that becomes psychologically sticky.

Examples:

- reflective highlights
- headlights
- illuminated hands
- cigarettes
- signage
- silhouettes

## Goal

Preserve emotional retention.

## Failure mode

- over-emphasis
- artificial subject weighting

---

# Key Principle

TRACE should evolve through:

small believable tendencies
+
careful interaction between primitives

Not dramatic transformations.
