# TRACE Deepening: Anti-Phone Behavior

TRACE is not anti-technology.

TRACE is increasingly anti:

```text
perceptual over-correction
```

---

# Core Thesis

Modern phone imaging systems often assume:

- all information should be visible
- all shadows should be recoverable
- all lighting should normalize
- all environments should become legible
- all faces should remain readable
- all scenes should feel technically successful

But emotionally believable perception often depends on:

- ambiguity
- uncertainty
- incompleteness
- collapse
- uneven visibility
- informational restraint

---

# Anti-Phone Behaviors

## Informational Restraint

Not all details should survive.

Examples:

- swallowed corners
- unreadable distant backgrounds
- partially hidden faces
- incomplete rooms

Emotional effect:

- realism
- intimacy
- environmental truthfulness

---

## Mixed-Light Preservation

Color disagreement should sometimes remain unresolved.

Examples:

- sodium vapor + fluorescent conflict
- flash + neon spill
- warm interior + cold exterior

Emotional effect:

- environmental realism
- perceptual instability
- nightlife authenticity

---

## Spatial Incompleteness

Space should sometimes collapse naturally.

Examples:

- disappearing architecture
- detached foregrounds
- swallowed depth

Emotional effect:

- memory realism
- emotional tension
- anti-HDR behavior

---

## Imperfect Temporal Continuity

Sequences should not feel uniformly optimized.

Examples:

- subtle drift
- recurring darkness
- perceptual fatigue
- repeated anchor dependence

Emotional effect:

- lived-in realism
- environmental continuity
- emotional accumulation

---

## Emotional Ambiguity

Images should not explain everything.

Examples:

- uncertain gestures
- unclear spatial relationships
- partial environmental understanding

Emotional effect:

- observational realism
- perceptual honesty
- emotional openness

---

# Important Principle

TRACE should not intentionally degrade images.

It should preserve:

```text
believable perceptual incompleteness
```

That is very different from:

- retro filters
- fake degradation
- artificial lo-fi aesthetics

---

# Review Questions

- Did the image preserve uncertainty naturally?
- Did the environment remain emotionally believable?
- Would a phone have revealed too much?
- Did ambiguity improve realism?
- Did the sequence feel perceptually honest?
