# TRACE Memory Anchors

Some visual details carry disproportionate emotional weight.

TRACE refers to these as:

```text
memory anchors
```

Examples:

- isolated highlights
- reflective surfaces
- cigarette glow
- headlights
- mirror reflections
- signage
- faces
- hands
- glass reflections
- flash-hot objects

---

# Core Principle

Memory anchors are not necessarily:

- the main subject
- the brightest object
- the sharpest region

Instead, they are:

```text
visually sticky details
```

that help an image persist emotionally.

---

# Why This Matters

Many emotionally believable flash photos contain:

- one glowing object
- one isolated face
- one reflective detail
- one harsh specular highlight

surrounded by:

- darkness
- ambiguity
- compression
- partial visibility

This contrast appears important for:

- memory pressure
- atmosphere
- emotional realism

---

# Current TRACE Approach

Current implementation is intentionally simple.

TRACE currently studies:

- isolated luminance clusters
- highlight concentration
- bright connected regions

This is:

- inspectable
- behavior-oriented
- lightweight

Avoid:

- heavy object recognition
- semantic tagging systems
- generic saliency models

---

# Future Directions

Potential future anchor studies:

- cigarette glow
- mirror flash duplication
- reflective asphalt
- signage persistence
- isolated skin highlights
- face-anchor weighting
- hand/object interaction

---

# Important Principle

Memory anchors should influence behavior subtly.

TRACE should not:

- artificially spotlight objects
- create cinematic focal points
- force dramatic composition

Instead:

anchors should help preserve how ordinary environments are remembered imperfectly.
